# CDTWEB — CodeDevTools Write • Edit • Backup

macOS desktop utility for structured code snapshotting + intelligent backups.

## Features
- **Smart Snapshot:** Capture the state of your codebase instantly.
- **Tree Export:** Export your project structure as a readable tree.
- **Clean External Backups:** Keep your backups organized and separate from your working directory.
- **Write from Blocks:** Assemble code from predefined blocks.

## Tech Stack
- Swift + SwiftUI
- macOS

## Folder Overview
- `CDTWEB/`: Main application source code.
- `Core/`: Core business logic, file operations, and backup services.
- `UI/`: Reusable UI components.
- `Views/`: Main SwiftUI views.
- `Toolbox/`: Sibling target/folder for additional utilities.

## Run/Build Instructions
1. Open `CDTWEB.xcodeproj` in Xcode.
2. Select the `CDTWEB` scheme.
3. Build and run the project (Cmd + R).
